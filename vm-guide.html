Troubleshooting

Common Issues

  • VM slow hai: Browser refresh kariye
  • Commands work nahi kar rahe: Course switch kar ke try kariye
  • AI commands not found: AI Security course select karo
  • ML datasets missing: ML Theory course mein switch karo
  • Files save nahi ho rahe: Browser storage clear kar ke try kariye
  • Mobile mein small text: Browser zoom use kariye

Reset Options

  • Terminal Reset: "Reset Terminal" button
  • Clear Screen: "clear" command ya "Clear Screen" button
  • Course Switch: Right side course shortcuts
  • Full Refresh: Browser refresh (F5)
